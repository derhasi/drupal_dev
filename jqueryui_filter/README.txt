This module implements jquery ui into input filters.

It is currently located on http://github.com/derhasi/dev/tree/jqueryui_filter and
was written by Johannes Haseitl - derhasi - http://derhasi.de

Dependencies:
- Jquery-UI with at least JqueryUI 1.7.2
- jquery_update-6.x-2.x-dev (1.x does not work)

Tabs
===========

At the moment only jquery ui Tabs is implemented. Therefore an additional wrapper
div embraces the text. Within this wrapper list items that point to a dependent
div are formed to tabs.
Like in jquery ui tabs the list items shall contain a link that points to the id
of the assigned tab:

Theme:
-----------
At the moment the base jquery-ui-theme will be used
if variable 'jqueryui_filter_use_jqueryui_css' is set to TRUE.

In generell the base theme  ui.tabs.css is implemented.
